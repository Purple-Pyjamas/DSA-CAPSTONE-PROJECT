# DSA-CAPSTONE-PROJECT

This repository contains two individual projects:

## 1. amazon-product-review-analysis
Unravelling price patterns of different variables across over a thousand Amazon products using excel, to help generate insights that can guide product improvement and marketing strategies, and enhance customer engagement.  


## 2. palmoria-hr-gender-analysis
Analyzing gender distribution, pay gaps, and performance ratings across Palmoria Group's 3 regions, using PowerBi, to reaveal insights on salary inequality, and help drive management action; Estimating the new salary for all employees after including their bonuses per rating.
